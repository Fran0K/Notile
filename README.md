# Notile

**Bring your notch to life**

A macOS menu bar reminder app that rolls out from the notch with a smooth blind-style animation.

<br>

[Download](https://github.com/Fran0k/Notile/releases) · [Report Issue](https://github.com/Fran0k/Notile/issues)

## What does it do?

Notile pops up reminders from your Mac's notch area with a smooth roll-down animation, helping you remember to:

- Rest your eyes
- Stand up and stretch
- Drink some water
- ...and anything else you want to be reminded of regularly

Reminders don't steal focus or interrupt your workflow. They collapse automatically when done.

## Features

- **Blind-style animation** — Expands from the notch with spring physics, collapses smoothly
- **Multi-reminder management** — Independent interval, duration, and animation for each reminder
- **Rich media support** — Lottie / PNG / GIF / JPEG files as reminder visuals
- **Custom display screen** — Choose which monitor to show reminders on
- **Custom content** — Title, message, and media for each reminder
- **Active hours** — Set reminders to only appear during specific time windows
- **Ready to use** — Built-in eye rest, hydration, and activity reminders
- **Multi-language** — English / 中文 / Français / 日本語, follows system language
- **Launch at login** — Optional auto-start via system login item

## Requirements

- macOS 14.7+
- Apple Silicon / Intel

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/Fran0k/Notile/releases)
2. Open the DMG, drag to Applications
3. Launch and find the menu bar icon, open Settings to get started
