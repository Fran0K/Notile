# Notile

**让灵动岛活起来。**

macOS 菜单栏定时提醒应用——以卷帘窗动画从刘海区域优雅弹出。

## 它做什么？

Notile 在你 Mac 的刘海区域以卷帘动画弹出提醒，帮你记住：

- 休息一下眼睛
- 站起来活动活动
- 喝杯水
- ……以及任何你想定期提醒自己的事

提醒弹出时不会抢走焦点，也不会打断你的工作流。看完自动收起，就像它从没来过。

## 特色

- **卷帘动画** — 从刘海展开、弹簧收起，流畅自然
- **多提醒管理** — 每个提醒独立设置间隔、时长、动画
- **丰富媒体支持** — 支持 Lottie / PNG / GIF / JPEG 文件作为提醒动画
- **自定义显示屏幕** — 选择在哪个显示器上弹出提醒
- **自定义内容** — 支持标题、正文，搭配动画或图片
- **启用时段** — 设定提醒只在特定时间段弹出，其余时间安静
- **开箱即用** — 内置护眼、喝水、活动三项提醒，装上就能用
- **多语言** — 中文 / English / Français / 日本語，跟随系统语言
- **开机自启** — 可选登录时自动启动


## 系统要求

- macOS 14.7+
- Apple Silicon / Intel

## 安装

1. 从 [Releases](https://github.com/Fran0k/NotchRoller/releases) 下载最新版 `.dmg`
2. 打开 DMG，拖入 Applications
3. 启动后在菜单栏找到图标，打开设置即可开始
